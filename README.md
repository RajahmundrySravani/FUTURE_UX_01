# Salon Website UI Design

## Project Overview
This project is part of Future Interns UI/UX Task 1.

## Pages Designed
- Homepage
- Service Page
- Contact Page

## Tools Used
- Figma

## Design Rationale
- I designed a clean and modern salon website focused on user experience  
- The homepage clearly shows services and includes strong call-to-action buttons  
- The “Book Now” button is placed prominently to improve conversions  
- The service page is structured with simple cards for easy understanding  
- The contact page includes a user-friendly booking form  
- The design is mobile-friendly and easy to navigate
